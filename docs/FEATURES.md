# Features

Complete feature list for HushBox, organized by category.

---

## Chat & Models

| Feature                   | Status  | Description                                                                                       |
| ------------------------- | ------- | ------------------------------------------------------------------------------------------------- |
| Multi-Model Chat          | Shipped | Access GPT, Claude, Gemini, Grok, and all models available through OpenRouter from one interface. |
| Model Switching           | Shipped | Change to a different AI model at any point, even mid-conversation.                               |
| Auto Model Selection      | Planned | Let the system choose the optimal model based on your task.                                       |
| Multi-Model Conversations | Planned | Chat with multiple AI models in the same conversation to compare responses.                       |
| Response Regeneration     | Planned | Generate a new response to the same prompt if you want a different answer.                        |
| Context Compacting        | Planned | When approaching context limits, compress the conversation history to continue chatting.          |

## Security & Encryption

| Feature                   | Status  | Description                                                                  |
| ------------------------- | ------- | ---------------------------------------------------------------------------- |
| Zero-Knowledge Password   | Shipped | OPAQUE protocol ensures your password is never transmitted to the server.    |
| End-to-End Encryption     | Shipped | Messages encrypted in-browser before storage. Server stores only ciphertext. |
| Recovery Phrase           | Shipped | 12-word BIP39 mnemonic backup for encryption key recovery.                   |
| Two-Factor Authentication | Shipped | TOTP-based 2FA with authenticator app support.                               |
| Rate Limiting             | Shipped | Auth endpoints protected against brute-force attempts.                       |
| Encrypted Sessions        | Shipped | Iron-session encrypted cookies with no server-side session storage.          |
| Pseudo-Anonymous Requests | Shipped | AI providers see HushBox credentials, not your identity.                     |
